Неделя 2: Создание чекпоинта

February 10, 2025

Что сделано:

Чекпоинт “Почтовая велопарковка” Мы создали атмосферный пункт сохранения в виде:

Старинного чугунного фонаря с витражными стеклами, излучающего теплый свет

Сказочного велосипеда Пикс, сочетающего:

✓ Раму классического байка 50-х годов
✓ Мотоциклетный руль с блестящими рычагами
✓ Турбо-насадки из латунных труб и шестеренок
✓ Кожаное седло с выгравированными созвездиями

К заднему колесу крепится сумка для писем-снов со:

  • Светящимися конвертами
  • Свисающими печатями в виде лун

Особенности дизайна:
• При активации фонарь зажигается голубым пламенем
• В воздухе появляются мерцающие частицы (как светлячки)
• Из сумки доносятся шёпот обрывков снов

Новый помощник: Хамелеон-фонарик Разработан альтернативный источник света:

Механика работы:
✓ Меняет цвет в зависимости от окружения
✓ Высвечивает скрытые символы на стенах
✓ Реагирует на приближение опасности (красное свечение)

Визуальные особенности:
✓ Полупрозрачная кожа с мерцающим узором
✓ Глаза-линзы, фокусирующие луч
✓ Хвост, выполняющий роль рукоятки

Игровая интеграция: Чекпоинт появляется:

  • После сложных платформенных участков
  • Перед битвами с мини-боссами
  • В местах важных сюжетных решений
